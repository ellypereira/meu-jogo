Política de Privacidade — Blood And Silence

Versão: 1.0 — Vigente desde 2 de dezembro de 2025.

Esta política explica como o jogo Blood And Silence coleta, usa, armazena e protege dados dos usuários. Leia com atenção.

1. Que dados coletamos

O jogo e o site podem armazenar e/ou processar os seguintes tipos de dados:

  • Dados locais (localStorage): progresso do jogo (capítulo/estágio), afinidades, nome do jogador, configurações (volume, preferências), flags de funcionalidades. Esses dados ficam salvos no navegador do usuário e não são enviados automaticamente a servidores.
  • Dados de autenticação anônima (Firebase, opcional): se a autenticação anônima do Firebase for usada, um identificador de usuário anônimo (UID) pode ser criado para salvar metadados no banco de dados Realtime Database ou Firestore; esses dados podem incluir apenas nome público e timestamp de criação, conforme configurado pela desenvolvedora.
  • Dados coletados por terceiros: serviços de anúncios (ex.: Google AdSense) e provedores de analytics (se integrados) podem coletar cookies, identificadores de publicidade, ou dados de uso agregados para personalização de anúncios e relatórios. Esses provedores têm suas próprias políticas.
  • Dados enviados voluntariamente: contatos via formulário, e-mail, ou informações que o jogador decida enviar manualmente para suporte/contato.

2. Para que usamos os dados

Usamos os dados para finalidades limitadas e legítimas:

  • Salvar e restaurar o progresso do jogo (localmente).
  • Oferecer preferências de usuário (volume, nome salvo, configurações).
  • Melhorar a experiência do jogo com análises agregadas e correções de bugs.
  • Monetização passiva por anúncios exibidos em páginas do site/blog (quando aplicável).
  • Comunicação com usuários, caso o jogador entre em contato via e-mail.

Não usamos dados para fins que não estejam listados aqui sem consentimento explícito.

3. Anúncios e serviços de terceiros

O site pode exibir anúncios através de redes de terceiros (ex.: Google AdSense). Esses provedores podem usar cookies, identificadores de dispositivo, e outras tecnologias para veicular anúncios relevantes. A desenvolvedora NÃO controla diretamente as práticas desses provedores.

Importantíssimo: não solicitamos nem incentivamos cliques em anúncios. Oferecemos conteúdo editorial e opções de apoio (Patreon/Ko-fi) como forma segura de apoiar o projeto.

Para saber mais sobre como o Google usa dados para anúncios, visite as páginas oficiais do Google Ads/AdSense e a Central de Privacidade do Google.

4. Como armazenamos dados

LocalStorage (no navegador): progressos e preferências ficam salvos localmente no dispositivo do jogador. Para apagar, o usuário pode limpar o armazenamento local do navegador (devtools → Application → Local Storage → remover) ou usar a opção de exclusão disponível no jogo, se houver.

Firebase / servidor: se configurado, alguns dados mínimos (ex.: nome público e UID anon) podem ser gravados em Realtime Database ou Firestore. Estes dados são armazenados nos servidores da Google (ou outro provedor) e estão sujeitos à política de privacidade do provedor.

5. Por quanto tempo retemos dados

Dados locais permanecem no dispositivo até serem removidos pelo usuário ou pelo navegador. Dados armazenados em serviços de backend (ex.: Firebase) são mantidos conforme a configuração da desenvolvedora; caso deseje exclusão, solicite por e-mail (veja seção de contato).

6. Seus direitos (GDPR / CCPA e similares)

Dependendo da sua jurisdição, você pode ter direitos como:

  • Solicitar acesso aos dados que mantemos sobre você;
  • Solicitar correção ou exclusão de dados;
  • Retirar consentimento quando aplicável;
  • Solicitar a portabilidade dos dados (quando possível).

Para exercer qualquer direito, envie uma solicitação para o e-mail de contato indicado abaixo. Tentaremos responder dentro de um prazo razoável (normalmente 30 dias).

7. Segurança

Tomamos medidas razoáveis para proteger dados (criptografia em trânsito para servidores, boas práticas de codificação). Entretanto, nenhuma transmissão na Internet ou armazenamento digital é 100% seguro. Se ocorrer um incidente relevante, comunicaremos rapidamente aos afetados quando requerido por lei.

8. Público infantil

O jogo não é direcionado a crianças pequenas. Não coletamos intencionalmente informações pessoais de menores de idade. Se souber que uma criança forneceu dados pessoais sem consentimento dos pais, entre em contato para que possamos remover os dados.

9. Alterações nesta política

Esta política pode ser atualizada. A versão vigente e a data de atualização estarão no topo da página. Recomendamos que verifique periodicamente.
